How many elements are there in each of the following
vectors? What are the values of the elements?
(a) vector<int> v1; - 0 elements.
(b) vector<int> v2(10); - 10 elements, value = 0.
(c) vector<int> v3(10, 42); - 10 elements, value = 42.
(d) vector<int> v4{10}; - 1 element, value = 10.
(e) vector<int> v5{10, 42}; - 2 elements, values = 10, 42.
(f) vector<string> v6{10}; - 10 elements, value = "".
(g) vector<string> v7{10, "hi"}; 10 elements, value = "hi".
